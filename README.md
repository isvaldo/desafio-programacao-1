### Instalação

#### Dependências

Instalação no Ubuntu

```
$ sudo apt-get install python-dev
$ sudo apt-get install python-pip
$ sudo apt-get install unzip
$ sudo pip install pyheaderfile
```

Para instalar aplicação execute
```
make install
```
Para iniciar aplicação execute 
```
make run
```
se precisar remotar
```
make uninstall
```
